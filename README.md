### matrix-site 
This is a matrix code with falling from above the lines of code.  
Enter an input line where the characters you type turn into words that fall from above.

#### How to use
- install nginx  
  `sudo apt install nginx`
- download all files  
  `git clone https://github.com/Pakdelark/matrix-site.git`
- place the downloaded files in the directory /var/www/html/  
  `mv index.html matrix.js /var/www/html/`
